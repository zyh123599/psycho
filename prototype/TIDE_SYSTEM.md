# 心潮 V0.6 潮向、潮笺与画像边界

四股潮向为选择提供即时叙事反馈；潮笺把一次满潮变成可回顾短句。它们服务于“观察当下”，不是心理测评、健康指标、疗效指标或人格画像。

## 四股潮向

| 潮向 | 界面含义 | 满潮短句语气 |
| --- | --- | --- |
| 觉察 `◐` | 辨认念头、感受与正在发生的模式 | 观察、命名、拉开一点距离 |
| 安定 `⌁` | 为身体、边界与当下留出落脚处 | 暂停、允许、回到此刻 |
| 联结 `∞` | 与他人及内在不同声音保持联系 | 真实表达、被听见、内在对话 |
| 精力 `✦` | 照顾可用力气，也允许疲惫存在 | 小步行动、保留力气、不必耗尽 |

每章从同一固定基线开始。左右选择和“两个都不像”只触发 `app.js` 中固定的 `tideEffects`：界面不展示增量数字、精确值、总分、排名、长短板或雷达图。满潮不是答对、升级或治疗进展；每股潮向每章最多触发一次，刷新或新开章节后清空。

## 模型叙事与本地规则

自定义模型可以结合确认主题与本机多模态文字画像，生成六张卡的角色、场景、问题和选择文案。六个位置的选择语义由本地约束，实际信号、潮向、阈值与收藏逻辑仍由本地代码附加。

模型生成失败、未授权或网络不可用时使用六张内置卡。用户开始答题后，迟到的异步响应不得替换正在阅读的卡片。左右选项都必须像真实选择并各自具有保护作用，不能设置明显“健康正确答案”。

## 潮笺存储

满潮后，用户可主动把一张内置短句收进卡槽。本机只保存卡片 ID 和收藏时间：

```json
{
  "id": "grounding-1",
  "collectedAt": 1784361600000
}
```

不保存短句副本、章节闪念、选择、潮位、聊天或画像。同一卡片自动去重，用户可在「我的」中清空。

## 日报和初印象

- 未启用持续画像时，只用当前页面中的闪念文字低粒度主题与近 30 天主动收藏的潮笺类型；
- 启用后，可使用本机画像中的标题、需要、资源、行动与反思问题；
- 图片只通过模型形成文字化观察，日报和初印象不会重新上传原图；
- 数据不足时显示基础版，不假装已经了解用户；
- “有帮助 / 不太像我”只评价内容，不是心理得分，也不反向改变潮位；
- 不显示分数、人格结论、诊断或“变好/变差”。

## 信号隔离

| 数据 | 本章潮位 | 对话/叙事 | 日报/初印象 | 默认持久化 |
| --- | --- | --- | --- | --- |
| 情境选择 | 是 | 结构化汇总 | 否 | 否 |
| 四股潮向 | 是 | 否 | 否 | 否 |
| 已确认主题 | 否 | 本次授权后可用 | 否 | 否 |
| 随手闪念文字 | 否 | 用户主动进入章节后可用 | 低粒度主题 | 否 |
| 随手闪念图片/语音演示 | 否 | 当前不发送 | 只使用“采用过该表达形式” | 否 |
| 章节授权图片 | 否 | 形成多模态文字画像后可用 | 只使用文字观察 | 否 |
| 聊天原文 | 否 | 当前会话；授权后可更新画像 | 否 | 否 |
| 主动收藏潮笺类型 | 否 | 默认不用 | 是 | 本机至用户删除 |
| 主动保存未来回响 | 否 | 否 | 否 | 本机至用户删除 |

聊天原文按次直达用户自定义服务商；本机画像只保存脱敏结构化结果。模型服务授权不等于画像授权。撤销画像授权会停止更新，删除画像会取消在途请求并清除画像，但不会自动删除潮笺或未来回响。

## 安全边界

- 潮向、卡片、日报、初印象和画像不得被表述为诊断、治疗、人格判断或正确答案；
- 不从脸、表情、身体、衣着或外貌推断心理状态或敏感属性；
- 发现自伤、伤人或即时危险线索时，暂停游戏化普通内容，优先显示所在地紧急服务、危机资源与可信任真人支持；
- 正式上线前需完成心理专业、内容安全、隐私合规、未成年人保护、地区危机资源和无障碍评审；
- 自定义模型接口与 CORS 要求见 [`CUSTOM_API.md`](CUSTOM_API.md)。
