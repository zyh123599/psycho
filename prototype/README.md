# 心潮 V0.5：多模态持续画像与情绪陪伴

「心潮」把每日低负担阅读、Reigns 式叙事选择和可随时结束的情绪陪伴串成一段连续体验。四股潮向在章节中可见，但章节结尾不计算总分、排名、人格结论或“治疗效果”。

当前交付物是可在浏览器和 Capacitor 运行的移动端应用前端，已连接仓库内 FastAPI 后端。它不是 Figma 源文件。

## 本地预览

先启动后端：

```bash
cd backend
uv sync --all-groups
uv run uvicorn psycho_backend.main:app --reload --host 127.0.0.1 --port 8000
```

再从仓库根目录启动前端：

```bash
npm install
cp prototype/.env.example prototype/.env
npm run dev
```

浏览器开发默认连接 `http://127.0.0.1:8000`。Capacitor 真机和生产构建必须通过
`VITE_API_BASE_URL` 配置设备可访问的 HTTPS 后端；任何上游模型密钥都只能放在 `backend/.env`。

## V0.5 体验结构

### 首页与自由探索

- **今日潮汐**：先给出一句今日判断、两条可解释依据、三条低负担建议和一句原创短句。没有可用信号时明确显示“基础日报”。
- **答案之书**：用户在心里保留问题，每天抽取一张本地原创答案卡；应用不收集问题，当天不能重复抽取。
- **随手闪念**：从底部导航进入后显示为一叠真实卡组，当前卡完整浮在最上层，后续卡从边缘露出；点击后层或使用左右按钮可以换到另一张。底部采用带胶带、横线和折角的统一便签编辑器，可以在同一张便签中组合文字、图片和语音；图片卡采用照片叠加倾斜信息纸样式，语音以波形和时长显示。
- **雨声背景音乐**：顶部「雨声」按钮由用户主动开启，使用 `assets/sleepy-rain.mp3` 低音量循环播放；再次点击暂停，不尝试绕过浏览器自动播放限制。
- **自由对话**：用户可以从底部导航或日报进入“潮伴”，使用快捷表达、文字、语音输入演示或语音对话演示，随时结束并返回首页。
- **今日章节**：保留完整叙事主线，适合愿意多花几分钟梳理一条具体线索的用户。
- **未来回响**：主页提供常驻入口，不再只依赖到期提醒或章节结束后的入口。

底部导航精简为「今天 / 闪念 / 对话 / 我的」。潮笺卡槽移入「我的」，对话图标与其他入口保持同一高度。

### 今日章节主线

1. **闪念便贴**：写下 2–4 个当下掠过脑海的念头，并决定它们是否参与本次梳理。
2. **主题确认**：系统提出临时候选；主题必须由用户确认或改写，不生成永久人格标签。
3. **情境选择与四股潮向**：通过六张 Reigns 式情境卡探索同一线索。选择会实时改变「觉察、安定、联结、精力」四条无数字潮位。
4. **潮笺收集**：任一潮向满格后浮出一张原创短句卡。「收进卡槽，继续」会收藏内置卡片；「让它经过」不收藏。两种选择都继续主线，也都没有惩罚。
5. **情绪陪伴对话**：完成情境卡后进入开放对话。开场会承接本次已确认主题与少量结构化选择线索；用户可纠正、跳过、自由输入或立即结束。它替代 V0.3 的固定“三轮回应”。
6. **微行动**：从几件现实中可完成的小事里选择一步，也可以跳过；没有打卡和连续完成压力。
7. **未来回响**：本章收进卡槽的短句可成为候选，也可选择其他句子或自己写。只有主动勾选后，文字与解封日期才会保存到当前浏览器。
8. **完成页**：回顾本次主题、微行动、未来回响和已收藏潮笺，不展示四维分数、优劣判断或长期画像。

## 日报与画像驱动内容

- 用户未开启持续画像时，日报读取近 30 天用户提交的**闪念文字主题**和主动收藏的**内置潮笺类型**；没有记录时显示基础版。
- 用户单独开启持续画像后，最新结构化画像会驱动主题候选、微行动、未来回响和日报。所有内容仍是可修正建议，不是固定定义。
- 基础规则用可解释关键词把闪念映射到「觉察、安定、联结、精力」内容方向，再与潮笺类型共同选择内容变体；数量相同时以最近一次信号决定。
- 日报会说明当前使用的是基础潮笺规则还是本机结构化画像，并允许选择「有一点帮助 / 不太像我」。反馈只保存为低粒度本机值；开启持续画像后可进入下一次后台更新。
- 图片只在首次有权利确认的画像请求中发送；不会被日报重新上传，也不会根据脸、表情、穿着或外貌推断心理状态。

## 支持性 AI 对话

- “潮伴”在用户按次同意后调用 `/api/v1/companion/respond`。接口返回严格结构化的普通回复、章节卡或反思报告；模型服务不可用时界面会明确标注本地降级回复。
- 聊天内容只存在于当前页面内存；结束或离开自由对话时清除，不写入 `localStorage`。后端无会话数据库，并对响应设置 `no-store`。
- 持续画像授权与对话服务授权分开。只有前者开启时，客户端才会把最小结构化画像上下文发送给对话端点，并在对话里程碑后台刷新画像。
- 对话不限制固定轮数；用户可以随时点击结束。当前输入上限为 400 个字符。
- 语音输入按钮会演示“聆听—生成示例文字”的前端状态；语音对话按钮会打开独立的通话界面演示。两者都不会调用麦克风、录音或上传音频。
- 前后端都包含高精度明确危险用语升级和显式“我现在需要紧急帮助”入口。命中后不调用普通模型，暂停陪聊并提示现实支持路径；这仍不是临床危机评估系统。

## 持续画像的异步策略

- 画像端点不会阻塞章节操作。便贴、选卡第 3/6 次、对话里程碑、行动和完成页只提交后台刷新信号。
- 同一时刻最多一个画像请求；请求期间的新变化合并为下一次快照，避免重复调用与旧结果覆盖新结果。
- FastAPI 和 OpenAI SDK 使用异步 I/O，并通过并发信号量保护上游；服务端当前无数据库或任务队列。
- 本机只保存最新结构化画像，并在写入前移除模型证据中的观察原文和安全事件。原始便贴、图片及聊天不写入画像存储。

## 交互方式

- 点击或轻触按钮完成填写、确认和页面导航。
- 闪念页在同一张便签编辑器里组合文字、图片和语音；「＋」展开相册与拍照入口，再次点击麦克风结束录音。
- 顶部「雨声」按钮控制背景音乐的播放与暂停，播放状态通过文字、颜色和动态音柱共同呈现。
- 情境卡支持鼠标或触控左右拖动，也可直接点击左右选项。
- 使用键盘 `←`、`→` 完成左右选择，`↓` 选择「两个都不像」；可用 `Tab` 在可交互元素间移动。
- 新手说明、语音演示和潮笺弹层可通过界面按钮关闭；支持时也可按 `Escape` 退出。
- 页面按移动端优先设计，也可在桌面浏览器中体验。

## 数据与隐私

- 章节中的闪念、确认主题、情境选择、章节潮位、聊天、微行动和未保存的未来回响只在当前页面内存中使用。
- 独立「闪念」入口提交后即视为收录，并用于更新「初印象」、日报方向和后续个性化内容；用户仍可查看、单条删除，或主动选择一条进入章节继续梳理。
- 「答案之书」只在本机保存当日日期和抽中的内置卡片序号，不收集用户在心里想的问题。
- 「我的 / 初印象」会参考闪念文字中可解释的近期主题、表达形式、主动收藏的潮笺类型和回响状态；不显示心理分数，也不把推断写成固定人格标签。
- 四股潮向在游戏中以无数字进度条展示；刷新页面或开始新章节后归零，不形成长期雷达图。
- 选择「收进卡槽」后，本机只写入内置潮笺编号和收藏时间；不保存短句副本、闪念、主题、选择、潮位、聊天或画像。同一卡片自动去重。
- 只有用户在「未来回响」中主动勾选保存时，回响文字和解封日期才写入浏览器 `localStorage`。
- AI 服务与持续画像分别授权，默认均关闭。设置页可随时关闭请求或删除本机结构化画像。
- 开启持续画像后会额外保存最新的脱敏结构化画像；它不会云同步，删除浏览器站点数据也会移除。
- 卡槽与未来回响均可在产品内清除；清理浏览器网站数据也会移除它们。
- 当前没有账号、云同步或服务端画像仓。授权的当次内容会由 FastAPI 转发给配置的第三方模型端点，服务端不持久化正文。
- 独立闪念中的图片只生成前端预览，语音只演示录音状态和时长；章节图片仅在持续画像授权和图片权利确认后按次发送。

生产接入所需的接口、数据分层、保留与删除规则见 [BACKEND_CONTRACT.md](./BACKEND_CONTRACT.md)。

## 产品与安全边界

“潮伴”的定位是**情绪陪伴和自我梳理伙伴**，不是“AI 治疗师”。卡片、四股潮向、日报和对话不能用于：

- 心理诊断、人格判定、风险证明或疗效评估；
- 替代心理咨询、医疗服务或现实中的人际支持；
- 保证用户会变好，或把满潮、收藏、连续使用包装成康复进度；
- 在用户表达即时危险时继续游戏化、奖励、泛化建议或普通角色扮演。

若用户或他人正处于即时危险，应暂停体验并联系所在地紧急服务、经核验的危机支持资源或能够立即到场的可信任对象。正式上线前仍需心理专业、危机安全、内容治理、隐私合规、未成年人保护和无障碍专项评审。
